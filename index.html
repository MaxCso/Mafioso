🎭 Les Rôles

MAFIEUX : La nuit, les mafieux se reconnaissent. Le jour, ils doivent persuader les autres de leur innocence.

👮 Police

POLICIER : Citoyen honnête. Le jour, il doit démasquer les mafieux par le débat et le vote.

🕵️ Infiltré

INFILTRÉ : Présent à partir de 6 joueurs. La nuit, il voit qui sont les mafieux, mais doit garder sa couverture.

🌙 Phase de Nuit

Tout le monde ferme les yeux. Le Maître du Jeu guide la nuit à voix haute. Les mafieux se reconnaissent, l'infiltré observe.

☀️ Phase de Jour

5 minutes de débat. Les joueurs votent pour éliminer un suspect. Le joueur éliminé révèle son rôle.

🏆 Victoire

Mafieux : Égaler ou dépasser le nombre de policiers.
Police : Éliminer tous les mafieux.

📊 Répartition

3-5 joueurs : 1 mafieux / reste policiers
6-8 joueurs : 2 mafieux / 1 infiltré
9+ joueurs : 3 mafieux / 1 infiltré
